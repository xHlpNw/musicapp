Аудиофайлы для начальных треков (data.sql)

Положите сюда MP3 (или другие аудио) с именами:
  sample1.mp3  — Come Together (The Beatles)
  sample2.mp3  — Something (The Beatles)
  sample3.mp3  — Time (Pink Floyd)
  sample4.mp3  — Bohemian Rhapsody (Queen)

При первом запуске приложения они будут скопированы в storage/tracks/,
и треки из data.sql смогут их проигрывать.

Можно использовать любые свои файлы с этими именами или переименовать
файлы в data.sql под свои имена.
